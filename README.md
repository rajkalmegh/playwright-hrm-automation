## Overview
This project demonstrates an end-to-end test automation framework built using Playwright and JavaScript to validate a web-based HR application.

## Tech Stack
- Playwright
- JavaScript
- Node.js
- Playwright Test Runner

## Test Coverage
- Login functionality
- Dashboard validation
- User navigation
- API validation using Playwright

## Project Highlights
- Page Object Model design
- UI and API automation
- Scalable folder structure
- HTML test reporting

## How to Run
1. Clone the repository
2. Run `npm install`
3. Execute tests using `npx playwright test`
4. View report using `npx playwright show-report`
